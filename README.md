# blogsitem  
*Basit bir arayüzle kendi blog sitemi oluşturdum.*

## 📝 Proje Hakkında  
Bu proje, kişisel blog sitesi ihtiyacımı karşılamak üzere geliştirilmiş sade ve işlevsel bir web projesidir. HTML, CSS gibi temel teknolojilerle hazırlanmış arayüz, ayrıca PHP ya da Python kullanımı da bulunabilir; dosya yapısındaki “mail.php” gibi öğelere bakılırsa iletişim formu için PHP’ye yer verilmiş.  
Genel olarak amaç: **kendine ait bir blog sitesi oluşturmak**, içerikleri kolayca yayınlamak ve paylaşmak.

## 🚀 Özellikler  
- Temiz, minimal ve kullanıcı dostu bir ön yüz arayüzü.  
- Hakkımda, İletişim, Blog gibi sayfaların hazır şablonları: `hakkımda.html`, `iletisim.html`, `blog.html` vb.  
- `mail.php` dosyası sayesinde iletişim formu yoluyla mesaj alınabilir.  
- Stil dosyası `style.css` ile temel görsel düzen sağlanmış.  
- Proje görselleri: logo ve afiş gibi çoklu png/jpeg dosyaları yer alıyor.  
- Front‑end ağırlıklı ama arka uç için de PHP ya da Python destekleri mevcut.

## 🧩 Dosya ve Klasör Yapısı  
- `blog.html` – Blog sayfası şablonu.  
- `hakkımda.html` – “Hakkımda” sayfası.  
- `iletisim.html` – İletişim sayfası.  
- `mail.php` – İletişim formu işleme scripti.  
- `style.css` – Tüm sayfalara uygulanan genel stil dosyası.  
- Görseller klasörü ya da doğrudan kök dizinde: `logo.png`, `acenta.png`, `afiş.png`, `site.png`, vs.  
- `app.py` – Eğer varsa Python ile çalışan bir backend betiği.  
- `proje.html`, `site.html` – Ek sayfa örnekleri ya da demo sayfaları.

## 🛠 Teknolojiler  
- HTML5  
- CSS3  
- PHP (iletişim formu için)  
- Python (opsiyonel/back-end için)  
- Statik site yaklaşımları için ideal.
